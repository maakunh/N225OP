## 1. そもそもオプションとIVとは

オプションは「ある価格（権利行使価格）で、将来売買できる権利」を売り買いする商品です。買う権利を**コール**、売る権利を**プット**と呼びます。日経225ミニオプションは、日経平均（正確には日経225ミニ先物）を対象にした、取引単位の小さいオプションです。

初心者がまず押さえるべきは、オプションの値段（プレミアム）が2つの要素で動くということです。ひとつは日経平均が上がった・下がったという**方向**。もうひとつが、これから相場がどれだけ大きく動きそうかという**期待のブレ幅**です。この「ブレ幅の期待値」を数値化したものが **IV（インプライド・ボラティリティ）** です。

### 基本用語

| 用語 | 意味 |
|---|---|
| 権利行使価格 | その権利が行使できる価格。「ストライク」とも。例：70000円のコール。 |
| プレミアム | オプションそのものの値段。ダッシュボードの「現在値」がこれ。 |
| IV | 市場が織り込む将来の変動率（年率%）。大きいほど「荒れる」と見られている。 |
| ATM | At The Money。今の日経平均にいちばん近い権利行使価格。最も活発に取引される。 |
| OTM | Out of The Money。今は権利行使しても得しない、価格的に外れた行使価格。 |

> **なぜ価格でなくIVを見るのか**
> 同じ「日経平均が上がると思う」でも、IVが高い（オプションが割高）ときに買うと、予想が当たっても利益が出にくいことがあります。IVは「オプションの割高・割安」を測る体温計です。だからこのダッシュボード群はIVを主役にしています。

## 2. IVスマイルの読み方

横軸が権利行使価格、縦軸がIVのグラフです。多くの場合、両端（高い行使価格と低い行使価格）でIVが高く、真ん中（ATM付近）で低い、笑った口のような曲線になります。これが「スマイル」の由来です。

### まず形を見る

日経のような株価指数では、左右対称の笑顔ではなく、**低い行使価格（下方）ほどIVが高い右肩下がり**になりがちです。これは「暴落への備え（プット買い）」の需要が常にあるためで、**スキュー（歪み）** と呼びます。下方のIVがATMよりどれだけ高いかを示すのが、画面上部の「下方スキュー」カードです。

| 見えること | 意味 | ヒント |
|---|---|---|
| 下方スキューが大きい | 下落への警戒が強い | 弱気・ヘッジ需要 |
| 曲線全体が上に持ち上がる | 相場全体が荒れる予想 | IV上昇 |
| 曲線全体が下に沈む | 市場が落ち着いている | IV低下 |
| ATM IVが低い | 当面は小動き予想 | 凪 |

### 時間で動かす

時刻スライダーを動かすと、スマイルが時間とともにどう変化したかが見えます。「始点と比較」にチェックを入れると、最初の時点（灰色の点線）と今を重ねられます。**曲線が上に移動していれば、その間にIVが上がった＝オプションが買われて割高方向に動いた**ということです。

> **OTM側IV（推奨）を使う理由**
> ATMより上はコール、下はプットのIVを採用すると、最も信頼できる滑らかなスマイルになります。深く外れたオプションは取引が薄く、コールIVが時々跳ねるためです。チェックを入れて使うのがおすすめです。

## 3. 価格/IVマルチチャートの読み方

権利行使価格ごとに、コール（青）とプット（桃の点線）の時間推移を小さなチャートで並べたものです。ATMの行使価格は枠が青くハイライトされます。「多数の値動きを一目で」見るための画面です。

### 価格とIV、2つのモードを切り替える

画面上部の **「価格／IV」トグル** で、格子状に並んだすべてのチャートの中身をワンクリックで切り替えられます。

- **「価格」モード**：各セルがコール価格・プット価格（プレミアム）の推移を表示します。いくらで売買されているか、絶対額の動きを見るモードです。
- **「IV」モード**：同じ格子のまま、各行使価格のコールIV・プットIVの推移に切り替わります。縦軸の単位は%になり、凡例も「コールIV／プットIV」に変わります。各セル右上の最終値ラベルやマウスオーバー時の数値にも `%` が付きます。

価格は「いくらか」、IVは「割高・割安か」を表します。**同じ並びのまま視点だけを切り替えられる**ので、「この行使価格、価格は上がっているがIVはどうか」をすばやく往復して確認できます。

### 同時に動くか、逆に動くか（価格モード）

日経平均が上がると、コール価格は上がり、プット価格は下がるのが基本です。多数のセルを一望して、**青(コール)が全体的に上向き／桃(プット)が下向きなら、相場は上昇局面**と読めます。逆に桃が一斉に立ち上がってきたら、下落が進んでいるサインです。

| 多くのセルで… | 相場の状態 | |
|---|---|---|
| 青(コール)が上昇 | 日経が上がっている | 上昇 |
| 桃(プット)が上昇 | 日経が下がっている | 下落 |
| 青も桃も上昇 | 方向は不明だがIV急騰（荒れ） | 変動拡大 |
| 青も桃も横ばい | 動意薄・様子見 | 小動き |

### 行使価格ごとのIVを横並びで比べる（IVモード）

IVモードに切り替えると、IVスマイル（第2章）が「ある瞬間の形」を見せるのに対し、こちらは**各行使価格のIVが時間とともにどう動いたかを、行使価格ごとに分けて**見られます。たとえばATM近辺だけIVが先行して上がっているのか、下方（プット側）の行使価格がそろって上昇しているのか、といった「どこのIVが動いているか」を格子の並びで把握できます。**価格モードで青・桃が逆向きでも、IVモードに切り替えると両方そろって上昇している**――そんなときは、方向というより相場全体の警戒感（IV）が高まっているサインです。

> **「0」は値段ではなく「取引が無い」**
> 線が途切れているのは、その時刻にその行使価格の取引・気配が無かったためです（価格・IVどちらのモードでも同じ挙動です）。途切れの多いセル（深く外れた行使価格）は流動性が低く、実際に売買するとスプレッド（売値と買値の差）が広くて不利になりがちです。**線が連続しているATM周辺ほど取引しやすい**と覚えてください。

## 4. IVヒートマップの読み方

縦に権利行使価格、横に時刻を並べ、IVの高さを色で表します。青が低い、赤が高い。**どの行使価格のIVが、いつ動いたか**を、色の面でまとめて掴むための画面です。

### 縦の色と横の色

縦方向（同じ時刻で上下）に色を見ると、スマイルと同じ情報＝どの行使価格のIVが高いかが分かります。横方向（同じ行使価格で時間が進む）に色の変化を見ると、**時間とともにIVが上がった（青→赤）か、下がった（赤→青）か**が分かります。

| 色の見え方 | 意味 | |
|---|---|---|
| 右へ進むほど赤くなる | IVが上昇中。警戒や思惑が高まっている | IV上昇 |
| 右へ進むほど青くなる | IVが低下中。相場が落ち着いてきた | IV低下 |
| 下半分が赤く偏る | 下方IVが高い＝下落ヘッジ需要 | スキュー |
| 急に一列だけ赤が走る | その時刻に何か（ニュース等）が起きた | イベント |

> **ヒートマップが得意なこと**
> スマイルは「ある瞬間の形」、ヒートマップは「時間を通じた変化」が得意です。寄り付きから引けにかけてIVがじわじわ上がったのか、特定の時刻に跳ねたのかは、ヒートマップの横の色変化が最も分かりやすく教えてくれます。

## 5. 読んだシグナルを売買に結びつける

ここからは、読み取った状態を実際の売買方針にどうつなげるかの考え方です。オプションには「買い」と「売り」があり、**IVが安いと思えば買い、高いと思えば売り**が基本の発想になります。

### IVの水準で方針が変わる

| ダッシュボードの状態 | 解釈 | とりうる方針の例 |
|---|---|---|
| スマイル全体が低い／ヒートマップが青い | オプションが割安 | 買い戦略が向きやすい（コール買い・プット買い等） |
| スマイル全体が高い／ヒートマップが赤い | オプションが割高 | 売り戦略が選択肢に（ただしリスク大、上級者向け） |
| 下方スキューが急拡大 | 下落不安が急増 | プット買いでヘッジ、または逆張りの判断材料 |
| 価格/IVマルチ（価格）で青が一斉上昇 | 上昇トレンド進行 | 押し目でのコール買いを検討 |

> **「売り」は初心者には勧めません**
> IVが高いときのオプション売りは、当たれば手堅い反面、相場が急変すると損失が理論上とても大きくなります（特にコール売り・裸売り）。まずは損失が払ったプレミアムに限定される**「買い」から**慣れるのが安全です。

### 3画面を組み合わせた判断の例

1. **ヒートマップで全体の温度を見る** — 今日はIVが上がっている局面か、下がっている局面か。買い場(青)か売り場(赤)かの当たりをつける。
2. **スマイルで歪みと水準を確認する** — 下方スキューの大きさで弱気の度合いを測り、ATM IVで「割高・割安」を具体数値で押さえる。
3. **価格/IVマルチで実際に売買する行使価格を選ぶ** — 線が連続している（＝流動性のある）ATM近辺の行使価格から、方向観に合うコール/プットを選ぶ。トグルをIVに切り替えれば、候補の行使価格のIVが高止まりしていないか（割高でないか）もその場で確認できる。

> **「予想が当たる」と「儲かる」は別**
> 方向が当たっても、買った時のIVが高すぎると利益が削られます。エントリー前に必ずIVの水準（スマイルの高さ・ヒートマップの色）を確認する癖をつけると、勝率と収益が噛み合いやすくなります。

## 6. 毎日の見る順番（ルーティン）

慣れるまでは、この順番で3画面を巡回するのがおすすめです。「全体→形→個別」と視点を絞り込んでいく流れです。

1. **ヒートマップ｜きょうの地合い** — 青寄りか赤寄りか。前日からIVが上がったか下がったか。
2. **IVスマイル｜歪みと水準** — 下方スキューとATM IVを数値で記録。時間スライダーで日中の変化も確認。
3. **価格/IVマルチ｜売買対象の選定** — 流動性のある行使価格で、方向観に合うものを絞る。トグルをIVに切り替え、選んだ行使価格が割高でないかも確認。

> **記録をつける**
> 毎日「ATM IV・下方スキュー・地合いの色」をメモするだけで、IVの高い・低いの自分なりの基準（平常時はIV何%くらいか）が身につきます。これが割高・割安判断のいちばんの土台になります。
